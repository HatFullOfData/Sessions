# Let's Build a Data Agent

## Description

In this almost no slides demo session, we'll have a play with turning a Microsoft Fabric semantic data model into a conversational data agent that your business users can actually chat to.

Starting from an existing data model, I'll show you how to ground an agent in your data, set up the instructions and example questions that shape how it behaves, and tweak things until it gives you proper answers instead of confident waffle. And how to turn a proof of concept into something that could be developed into part of your agentic architecture.
