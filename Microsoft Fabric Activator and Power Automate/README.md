# Microsoft Fabric Activator and Power Automate - a great combination

## Description

Awesome data tools within Microsoft Fabric means that anomalies, trends and repeating spikes can be detected promptly. Add in Activator and we can react with actions which includes Power Automate flows. This combination means Power Automate now has more data driven triggers in its tool box.

In this session we will start with the simple example and build up to a more complex business scenario to show off the possibilities Activator and Power Automate can perform together.

Lots of demos and resources to take away to have a go yourself
